Array of chemical objects JSON Example

This folder contains a chemicalTable.json example. Based on this information we should be able to:
  • Create a module that allows to display value in a table
  • Create a module that allows to display the content of a chemical object
  • With mouse over in the table show the result in the chemical module

The table module allows to display the content of the an array of object. We should be able to define as many columns as we want and to display the selected information in the column. The selection could be based on a kind of "XPATH" but applied to a json. Using a hierarchical drop down menu we would go through the object hierarchy to select the field to display.
